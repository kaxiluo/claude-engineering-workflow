# 项目配置

复制此文件为 `projects.md` 并填写你的项目信息：

```bash
cp references/projects.example.md references/projects.md
```

## 工作目录

所有项目所在的根目录：

```
~/workspace
```

## 项目列表

| 项目目录 | 说明 | Teambition 项目名（可选） |
|----------|------|--------------------------|
| my-project | 项目简要说明 | 对应的 Teambition 项目名称 |

## 排除项目

以下项目不参与月度总结等扫描类技能（仍可用于 Teambition 任务同步）：

| 项目目录 | 原因 |
|----------|------|
| my-demo | 临时测试项目 |
| archived-project | 已归档项目 |
